### INSTALACION REACT JS

Este proyecto se inició con Creat React App:

```sh
npx create-react-app ecommerce-my-app
cd ecommerce-my-app
```

Ejecución de la aplicación:

```sh
npm start
```

Creación de la carpeta 'build' para la puesta en producción

```sh
npm run build
```
